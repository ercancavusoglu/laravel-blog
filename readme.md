Blog Project with Laravel 5.4
======


>Installing
    
    #add vendors
    composer install
       
    #rename .env.examples to .env    
    #Run 
    php artisan key:generate
        
    #add migrations(entities) to database
    php artisan migrate
     
    #Add User    
    http://localhost:8000/register
                
    #run on the server
    php artisan serve
    
    That's it!
